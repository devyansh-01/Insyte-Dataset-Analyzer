# Insyte – Smart Dataset Analyzer

Insyte is a modern data analysis platform that transforms raw datasets into meaningful insights through automated statistics, data quality checks, interactive visualizations, and AI-powered recommendations.

## Features

### Automated Statistical Analysis
- Descriptive statistics
- Data profiling
- Numerical and categorical summaries
- Distribution analysis

### Intelligent Data Cleaning
- Missing value detection
- Duplicate record identification
- Data quality assessment
- Cleaning recommendations

### Interactive Visualizations
- Histograms
- Scatter plots
- Pie charts
- Correlation heatmaps
- Violin plots

### AI-Powered Insights
- Trend identification
- Correlation discovery
- Automated observations
- Dataset health insights

### Export & Reporting
- Excel export
- CSV export
- PDF report generation

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- OpenPyXL

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Insyte-Dataset-Analyzer.git
cd Insyte-Dataset-Analyzer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

## Project Goal

The goal of Insyte is to make data analysis accessible to everyone. Users can upload datasets and instantly receive statistical summaries, quality assessments, interactive visualizations, and actionable insights without extensive programming knowledge.
